VTT Captions Directory
=====================

This directory contains WebVTT (.vtt) caption files for videos.

Caption files should have the same name as the video file but with .vtt extension.

Example:
- Video: hospital-intro.mp4
- Caption: hospital-intro.vtt

WebVTT Format Example:
```
WEBVTT

00:00:01.000 --> 00:00:04.000
Chào mừng đến với Bệnh viện Đa khoa Mỹ Phước

00:00:05.000 --> 00:00:08.000
Chúng tôi cung cấp dịch vụ y tế chất lượng cao
```

For accessibility compliance, all videos should have corresponding caption files.
